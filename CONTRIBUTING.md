## 2. How Can I Help?

- **Feature Requests:** Open an issue to describe a skill discovery or RAG feature you'd like to see.
- **Bug Reports:** Submit an issue with a clear description and steps to reproduce.
- **Pull Requests:** - Fork the repo and create your branch from `main`.
    - Ensure your code follows the existing style.
    - Write clear commit messages.
    - Link the PR to any relevant issues.

## 3. Development Setup
