请始终用中文（简体）与我对话，除非我另行指示。

使用说明：
- 当你开启一个新的会话或向 assistant 发出第一条消息时，把本文件的全部内容粘到对话开头，或在会话第一条消息里简短说明“请用中文”。
- 这会使 assistant 在整个会话中优先使用中文回答，除非你在后续消息中明确要求其它语言。

可选偏好（你可以在此文件中添加）：
- 回答长度偏好（简短/详细）
- 是否在回答中包含代码示例
- 是否允许联网执行命令（如 git clone、npx）

示例：
1) 在新会话开始时，用户发送：
   请始终用中文（简体）回答本会话中的问题。

2) 若要允许联网安装技能，用户需明确回复：
   允许我运行 npx 和 git clone 来下载并安装技能（我知道这会联网执行外部命令）。
